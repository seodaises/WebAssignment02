## 📋 Project Overview
RoomSync is a dynamic, data-driven web application that helps users find verified rooms and compatible roommates across Pakistan. It was built as part of Assignment 02 for Web Engineering (Spring 2026) at The University of Lahore.
 
---
 
##  Key Features
- Full CRUD: Create, Read, Update, Delete room listings
- Search filter options (Verified, Shared, Private, Studio, Budget, Top Rated, Lahore)
- Dark/Light mode toggle with localStorage persistence
- 10 string methods used in search, formatting, and data cleaning
- Advanced array methods: filter, map, sort, reduce, find, every, some, includes
- Fully responsive with Tailwind CSS + Flowbite
- Semantic HTML5: nav, main, section, article, header, footer, aside
